# Gate 3：结果确认

完成求解后、开始论文写作前，必须确认结果真实、完整且可解释。

## 结果完整性

- 每一问是否都有可引用的结果？
- 结果是否来自 `results/` 或 `reports/RESULTS_REPORT.md`？
- 图表是否来自 `figures/`？
- 是否存在缺失结论、未解释异常值或未满足约束的结果？

## 合理性与稳健性

- 是否完成基线对比、约束检查或 sanity check？
- 是否报告敏感性、误差或不确定性分析？
- 随机算法是否记录随机种子和重复运行结果？
- 结论是否与数据、图表和模型输出一致？

## 人工确认

必须确认：

1. 不存在编造数值或未运行却声称完成的结果；
2. 每个最终结论均可追溯到结果文件；
3. 可以把确认后的结果用于论文。

未通过 Gate 3，不允许进入 Gate 4。
