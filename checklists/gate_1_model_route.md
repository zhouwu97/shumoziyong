# Gate 1：模型路线

进入建模细化前，必须确认模型路线，而不是只罗列模型名称。

## 路线闭合

- 每一问是否已经确定主模型和备用模型？
- 每个模型是否对应明确的输入、处理、输出？
- 模型之间的数据传递关系是否清楚？
- 是否说明为什么主模型比备选模型更适合？

## 数学表达

- 是否明确变量、参数、常量和单位？
- 是否定义目标函数、评价函数或判别函数？
- 是否列出约束条件及其来源？
- 是否区分题目给定条件和人工假设？

## 可实现性

- 数据是否足以支持模型？
- 算法复杂度是否适合比赛时间？
- 是否有基线方案或简单方法用于校验？
- 是否准备了失败降级路线？

## 人工确认

必须由人确认：

1. 子问题拆解没有偏题；
2. 主模型路线可接受；
3. 数据来源可获得；
4. 假设条件不过度；
5. 可以进入代码计划。

未通过 Gate 1，不允许进入 Gate 2。
