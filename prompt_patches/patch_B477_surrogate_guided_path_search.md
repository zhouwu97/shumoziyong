# patch_B477_surrogate_guided_path_search

- 类型：论文经验补丁草案
- 来源论文：2023 年高教社杯 B477《多波束测深合理探测方案的设计及效果分析》
- 适用题型：复杂地形覆盖 / 空间扫描路径规划 / 离散数据拟合 / 启发式搜索 / 分区优化
- 依赖：`prompt_base_v1.0.md`
- 进入 base：否
- 修改 plugin：否
- 初始状态：`review_ready`

## 职责边界

本补丁只记录 B477 的可迁移经验和误用风险，不重复 base 的通用规则，不负责旧题闭环，不负责 `competition_evidenced` 判定，不直接修改正式 plugin，不把单篇论文经验提升为通用规则。

## 核心经验

```text
先修正覆盖指标口径
-> 再把离散空间数据变成连续查询器
-> 再用局部几何变量约束路径搜索
-> 最后用分区、边界和失败路径检查结果
```

B477 的价值不在于随机森林或飞蛾火焰算法本身，而在于它把复杂曲面路径规划拆成了数据连续化、局部状态查询、几何方向选择、重叠约束计算和启发式搜索改造几个可检查环节。

## 可迁移经验

1. 真实空间数据通常是离散的，进入路径优化前应先判断是否需要连续代理模型。
2. 连续代理模型的输出必须服务后续建模变量，例如深度、坡度、梯度、成本、风险或容量，而不是只展示拟合效果。
3. 从规则模型迁移到真实曲面时，要说明哪些变量从常量变成位置函数。
4. 复杂曲面可先做局部微分近似，把小区域视为规则场景，再复用基础几何模型。
5. 路径方向不应只由算法随机搜索决定，应优先嵌入物理或几何含义，例如梯度正交方向、等值线方向、主轴方向或低风险方向。
6. 启发式算法需要为题目改造适应度函数，把覆盖率、重叠率、路径长度、边界漏覆盖和安全约束纳入评价。
7. 当统一搜索出现漏覆盖、路径振荡或鲁棒性差时，可以先分区，再在各区域内使用更稳定的局部策略。
8. 分区依据要来自数据结构或业务结构，例如等值线、梯度变化、空间密度、需求强度、风险等级或约束制度。
9. 复杂路径结果不要只给一张总图，应同时给分区统计表、总指标表、失败案例说明和边界处理说明。
10. 高拟合度、复杂算法和漂亮路径图都不能单独证明最优，需要参数对照、多起点或边界验算支撑。

## 适用条件

遇到以下题目时可以启用本补丁：

1. 题目给出离散空间数据，需要在连续区域内做路径、覆盖、巡检、扫描或服务布设；
2. 局部状态随位置变化，例如高度、深度、坡度、密度、风险、收益、容量或可达性；
3. 基础模型在规则场景成立，但真实场景是曲面、非均匀区域或复杂边界；
4. 需要用机器学习、插值或回归模型为优化器提供任意点查询；
5. 路径搜索算法需要嵌入覆盖率、重叠率、总长度、边界和分区约束；
6. 全局一次性搜索不稳定，或不同区域表现出明显不同的空间规律。

## 使用步骤

```text
1. 定义空间任务：覆盖、巡检、扫描、服务还是路径搜索？
2. 列出位置相关变量：哪些变量随坐标变化？
3. 判断数据形态：离散点是否足以直接计算？是否需要连续代理模型？
4. 建立代理模型：比较插值、回归或机器学习模型，并明确输出变量。
5. 接回基础模型：把代理模型输出输入到局部几何、成本或风险公式。
6. 确定局部方向：寻找能降低约束复杂度或提高覆盖效率的方向。
7. 改造搜索算法：把方向、覆盖、重叠、成本和边界写进适应度函数。
8. 做分区与边界检查：处理非均匀区域、交界区域和漏覆盖区域。
9. 输出证据表：总长度、覆盖率、漏覆盖率、超约束部分、分区统计和误用风险。
```


```text
在真实曲面中，……不再是常量，而是位置函数。因此本文将每个小区域近似为局部规则场景，用代理模型给出的……计算局部覆盖宽度和约束指标。
```

```text
路径搜索方向需要与空间结构相匹配。本文优先选择……方向，因为该方向可以……，从而降低后续重叠率或覆盖率控制难度。
```



## 误用风险

1. 不要因为论文使用随机森林，就在所有离散数据题中默认使用随机森林；模型应通过数据形态、样本量和验证指标选择。
2. 不要把训练拟合度高等同于路径规划可靠；边界区域、稀疏区域和外推区域需要单独检查。
3. 不要把启发式算法包装成全局最优；若没有严格证明，只能称为候选方案或近似方案。
4. 不要只给最终路径图；复杂路径规划必须报告覆盖、漏覆盖、超约束、总长度和分区指标。
5. 不要机械套用“沿等高线方向”或“梯度正交方向”；必须先说明该方向为什么能稳定覆盖宽度、降低成本或满足约束。
6. 不要忽略分区交界处；分区优化容易在交界产生漏覆盖、重复覆盖或路径连接成本。
7. 不要把单篇论文的算法细节、参数和结果直接写入 base 或正式 plugin。

## 不可照搬内容

以下内容只适用于 B477 或多波束测深背景，不能直接迁移：

1. B477 的多波束测深覆盖宽度、重叠率和跃迁公式；
2. 开角、坡度、水深、海域尺寸、测线条数、测线总长度、漏测率和超重叠长度等具体数值；
3. 随机森林在该附件数据上的高拟合结论；
4. 改进飞蛾火焰算法的具体螺旋函数、火源调整方式和参数；
5. 将测线方向固定为与梯度垂直或与等高线相切的结论；
6. B477 的七个分区、每区测线条数和“己”字形连接方式；
7. 用 10% 重叠率作为最终方案基准的具体取舍；
8. 附录 Python 代码结构和桌面文件路径。

## 与 base 的关系

base 已经负责题目理解、子问题拆解、题型判断、输入输出链、候选模型和人工确认。本补丁只在题目属于复杂空间覆盖、离散空间数据优化或路径搜索时追加以下检查：

1. 是否把离散数据转换为可查询变量；
2. 是否说明代理模型输出进入哪个公式或搜索算法；
3. 是否区分插值可视化和优化计算输入；
4. 是否说明局部变量如何从常量变成位置函数；
5. 是否把物理或几何方向写入搜索策略；
6. 是否对启发式结果做多起点、参数对照、失败路径或边界检查；
7. 是否处理分区交界风险；
8. 是否明确该补丁只是单篇论文经验，不进入 base，不直接标记 `competition_evidenced`。
