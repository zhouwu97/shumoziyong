## 对 AI 的强制要求

1. 不能直接套模型；
2. 必须先写输入、输出、变量、指标和约束；
3. 必须说明为什么这个模型适合本题；
4. 必须给出简单基准或降级方案；
5. 必须说明该补丁只提供启发，不等于通用规则。

## 推荐使用步骤

```text
1. 判断题型和触发条件
2. 抽象题目中的对象、变量、指标和约束
3. 选择可迁移经验
4. 排除不可照搬内容
5. 输出建模路线、验证计划和误用风险
```

## 可迁移表达模板

```text
本文首先将现实问题中的……抽象为……，并用……刻画方案优劣。
```

```text
在……明确后，本文将后续问题转化为以……为目标、以……为决策变量、以……为约束的数学任务。
```

## 不可照搬清单

1. 具体题目背景；
2. 具体数据结论；
3. 具体参数；
4. 只服务于这篇题目的公式；
5. 论文中为了表达美观但没有普适性的句式；
6. 没有解释为什么适用的高级算法；
7. 论文附录代码结构。
