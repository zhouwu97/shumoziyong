## 测试边界

本轮只验证多波束测线问题的基础代码骨架是否能跑通：

1. 题面固定参数；
2. 海里 / 米单位换算；
3. 覆盖宽度公式计算；
4. 重叠率计算；
5. 附件水深网格读取；

本报告不是最终测线方案，不写论文，不运行完整优化器，不使用遗传算法、粒子群、模拟退火或神经网络，也不沿用 2024-C 的地块、作物、季次、收益结构。
