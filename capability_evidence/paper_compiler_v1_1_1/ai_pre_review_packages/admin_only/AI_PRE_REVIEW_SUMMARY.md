# paper_compiler_v1_1_1 AI 探索性预评审汇总

本汇总只整合两份独立 AI 预评审，不重新评分，也不构成正式资格证据。两份结果的 `reviewer_type` 均为 `ai_pre_review`，不能替代两名外部真人评阅，不能据此宣称 `production_ready`。解盲映射为：X = A（现有固定模板），Z = B（论证图和段落规划），Y = C（论证图、段落规划和表达卡片）。

## 1. 评审完整性

两份 JSON 均通过 `paper_compiler_ai_pre_review.schema.json` 校验，且 `status` 均为 `completed`。两份结果都包含三个匿名版本、每版全部 11 项硬性检查、七个评分维度、结果定位时间、理解时间、修改时间估计、具体优点与缺点、三版本完整排序、全部强制比较问题、原文复核状态以及 `continue / revise / stop / inconclusive` 范围内的建议。因此两份结果均不标记 `review_incomplete`。

评阅人 1 按其包内规定记录 `X -> Y -> Z`，评阅人 2 按其包内规定记录 `Z -> X -> Y`，均与管理员文件中的预设顺序一致。两份 JSON 均声明 `independent_review: true`、`mapping_not_inferred: true`；管理员完整性报告还显示两包匿名材料一致、仅阅读顺序不同、包内不含映射及另一评阅人的材料，泄盲扫描均通过。现有可审计材料支持其遵守阅读顺序和独立盲评要求，但这些声明与包隔离不等同于对 AI 执行过程的法证证明。

两份结果均为 AI 预评审：评阅人 1 的原结论为 `ai_pre_review_continue`，评阅人 2 为 `ai_pre_review_revise`。该分歧在第 7 节汇总处理。

## 4. 结构规划信号

B 相对 A 显示一致且方向明确的探索性改善：两位评阅人都给出比较清晰度 +2、段落连贯 +1，归因质量和编辑就绪度也均提高；两人都把 B 排在 A 之前。理解时间分别减少 3 秒和 7 秒，预计修改时间均从 12 分钟降至 5 分钟。B 未报告硬错误，而 A 存在两人一致确认的无证据结论。

这些结果显示论证图和段落规划可能有效，尤其体现在显式呈现比较差额、形成“结果—比较—归因—边界”链条以及降低修改成本。不过，A 自身的硬错误会放大 B 相对 A 的优势；当前只有两份 AI 判断和一个试点文本，尚需外部真人验证，不能写成“已经证明结构规划有效”。

## 5. 表达卡片信号

C 相对 B 没有形成一致的净增益。评阅人 1 将归因质量提高 1 级并把 C 排在 B 前；评阅人 2 的归因评分不变并把 B 排在 C 前。两人都认为 C 的段落连贯和编辑就绪各低 1 级；理解时间分别增加 7 秒和 12 秒，预计修改时间均增加 3 分钟。结果定位、比较清晰和边界意识没有新增分值。

C 相对 A 的比较清晰和归因质量均获两人一致提高，但 C 未稳定优于 B，且理解时间反而比 A 长 4 秒和 5 秒。按照预设解释规则，主要增益更可能来自结构规划；表达卡片可能增强归因与边界的显式表达，但目前同时呈现重复、防御性句式、模板感或编辑成本增加的信号。该增量价值尚不稳定，需真人验证和更多样本复测。

## 6. 原文复核

自动事实验证方面，事实投影校验为 `passed`；B、C 的事实实现报告各检查 8 个绑定，均为 0 个失败、0 个问题。当前材料中没有与 B、C 同类的 A 版事实实现报告，因此不能把 B、C 的自动通过外推为 A 已自动通过。两位 AI 均人工式核对了 A 的数字、单位、精度、比较方向和求解边界，但同时识别出 A 的无证据结论。

自动原文重合检查的明确范围是 C（Y）和四张表达卡片。C 与来源的最长归一化连续匹配为 4 个字符，最高匹配是通用标题“结果分析”，字符 8-gram 重合率为 0.0，自动状态为 `passed`；四张卡片最长连续匹配为 2--3 个字符、8-gram 重合率均为 0.0，也均自动通过。两位 AI 均将总体状态判断为 `generic_academic_overlap`，三个版本的 `suspected_source_reuse` 均为 `false`。

自动通过不是人工证明，也不能覆盖未纳入自动比较的范围。`human_overlap_review.json` 仍为 `pending`，所有人工 verdict、comment 和 signature 均为空；因此真人原文复核尚未完成，不能形成正式无复用证明。

## 7. AI 预评审结论

`ai_pre_review_revise`

理由：结构规划路径显示较一致的探索性正向信号，B、C 也未被两位 AI 报告硬错误，因此不支持 `stop`；但 A 存在两人一致确认的无证据结论，因果越界分类还需真人或第三评阅人裁决，表达卡片增益也存在排名分歧与明确成本，故不能直接 `continue`。这一结论只要求修订和进一步验证，不改变项目资格状态。

`qualification_status` 必须保持 `awaiting_external_human_review`，不得改成通过；`production_allowed` 必须保持 `false`；不得根据本汇总宣布 `production_ready`。
