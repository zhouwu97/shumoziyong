# 2023-B 数学规格

## 坐标与基本几何

内部长度统一为米，深度向下为正。令半开角 `gamma=theta/2`。Q1 距离正向指向浅水侧：

```text
D(x) = 70 - x tan(alpha)
```

Q2 的 `beta=0` 沿坡面向上法向水平投影的反向，即深水方向：

```text
D(s,beta) = 120 + 1852 s tan(alpha) cos(beta)
delta = atan(tan(alpha) sin(beta))
W = D sin(gamma) [sec(gamma+delta) + sec(gamma-delta)]
```

`W` 是两个边缘波束与坡面交点之间的三维海底交线长度。Q1 相邻覆盖带宽度不同，重叠率使用
前一覆盖带宽度归一化的有符号交叠长度，负值表示漏测。

## Q3 决策规格

决策变量必须完整给出每条有限线段的两个端点、方向、顺序和有效长度。正式目标是所有裁剪后测线长度之和。
硬约束包括矩形边界内完全覆盖、相邻有效覆盖带重叠率处于 `[0.10, 0.20]`、几何量有限且水深为正。

## Q4 决策规格

输入只能是官方 `251x201` 水深网格。每条测线按二维地形上的局部水深、局部坡度、航向角计算覆盖带，
并裁剪到 `4x5` 海里边界。正式输出为完整测线、总长度、漏测面积占比和重叠率超过 20% 部分总长度。
局部平面仅是冻结尺度下的近似，不能在论文中宣称完整刻画高曲率三维声传播。
