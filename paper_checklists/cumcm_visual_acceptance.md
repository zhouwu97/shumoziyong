# CUMCM PDF 逐页视觉验收清单

## 使用方式

1. 先运行 `check_pdf_metadata.py` 和 `rasterize_pdf.py`。
2. 按页查看导出的 PNG，不跳页、不只抽查首页和末页。
3. 每个问题记录页码、严重级别、问题描述和状态。
4. 脚本通过不等于视觉通过；视觉结论必须由具备视觉能力的 AI 或人工给出。

## 严重级别

- `P0`：数字、公式、结论或关键内容缺失，不能提交。
- `P1`：表格、图片、正文、标题明显重叠或越界，必须修复。
- `P2`：可读性或一致性问题，建议提交前修复。
- `P3`：轻微美观问题，不影响理解。

## 全文级检查

- [ ] PDF 非空，页数与预期一致。
- [ ] 所有页面均为 A4，方向和尺寸一致。
- [ ] 没有意外空白页或重复页。
- [ ] 中文、英文和数学字体无明显缺字、乱码或异常回退。
- [ ] 页码连续、位置统一，未与正文或图表重叠。
- [ ] 黑白打印时标题、线型、图例和关键数据仍可区分。

## 每页检查

- [ ] 公式完整可见，编号、上下标、分式和矩阵未消失或裁切。
- [ ] 数学符号无方框、问号或替代字符。
- [ ] 中文无乱码，标点和英文混排正常。
- [ ] 正文、标题、页眉和页脚没有互相覆盖。
- [ ] 表格未超出页边距，单元格文字未折断、重叠或截断。
- [ ] 三线表只有必要横线，无表头底色和多余网格。
- [ ] 图片完整，坐标轴、图例、单位和标注可读。
- [ ] 图表没有被裁切、拉伸或压缩变形。
- [ ] 图题和表题未与图表或正文重叠。
- [ ] 图内没有与正文图题重复的大标题。
- [ ] 没有大面积无解释空白。
- [ ] 没有落在页底的孤立标题。
- [ ] 没有连续堆放图表而缺少正文解释。
- [ ] 图注长度适中，未挤压正文或跨页形成歧义。
- [ ] 彩色线型、区域或标记在灰度下仍可区分。

## 记录格式

```json
{
  "page": 8,
  "severity": "P1",
  "issue": "图注与横轴标签重叠",
  "status": "open",
  "reviewer": "human_or_visual_ai"
}
```

## 通过条件

- `P0` 和 `P1` 问题全部关闭；
- 所有页面均已人工或视觉 AI 查看；
- 未执行的检查明确记录原因；
- 自动报告、视觉记录和最终 PDF 指向同一文件 SHA。
