# 本轮闭环执行计划

## 1. 本轮目标

- 目标类型：优先执行 T3 泛化测试；如果本地和自动获取后仍无 T3/T4，则执行最高质量 T0/T1/T2 流程冒烟测试。
- 是否自动选题：是。用户提到 `B226.pdf`，但仍按规则扫描全项目并排序。
- 是否目标为 T3：是，优先尝试补齐 2023-B 或其他工程优化旧题官方材料。
- 是否允许计入 stable：本轮不正式标记 stable；只有累计至少 2 道不同题号 T3/T4 通过且无 P3/P4/P5/M1/M2/M3/M5，才建议 stable 候选。
- 本轮禁止事项：不写完整论文，不写求解代码，不给最终答案，不覆盖正式 base/plugin/patch，不把优秀论文 PDF 当官方题面。

## 3. 自动选题计划

- 扫描目录：`tests/old_problems/`、`output/closed_loop/`、`papers/`、`tmp/`、`高教社杯全国大学生数学建模竞赛优秀论文/`、项目根目录 PDF/DOCX/XLSX/CSV/TXT/MD。
- 候选题识别方法：按文件名、目录、已有记录、PDF 元信息和文本片段识别官方题面、附件、模板、优秀论文、解析、学习卡片。
- T0-T4 判定方法：官方题面/附件/模板完整性、是否答案泄漏、是否同题污染、是否已通过不同 T3。
- M1-M5 判定方法：优秀论文/解析泄漏、附件模板缺失、同题污染、经验照搬、人工补题意。
- 候选题排序表输出位置：`output/closed_loop/候选题排序_工程优化_2026-07-02.md`。
- 选题失败时动作：先自动获取官方材料；仍失败才生成失败报告和最小补材料清单。

## 4. 材料资格审查计划

| 检查项 | 判定方式 |
|---|---|
| 官方题面 | 来源 URL 或本地来源可追溯，非优秀论文/题解 |
| 官方附件 | 官方包或可信镜像附件完整 |
| 输出模板 | 官方结果模板、提交说明或题面明确输出要求 |
| 同题污染 | 与 A092/A127 2023-A 定日镜场同题则 M3 |
| 答案泄漏 | 优秀论文、获奖论文、解析、参考答案均触发 M1 |
| stable 资格 | 仅 T3/T4 且无 M1/M2/M3/M5，且不正式标记 stable |

## 5. 官方材料自动获取计划

- 是否需要自动获取官方材料：需要。当前日志显示本地无 stable 可计数 T3。
- 搜索目标：优先 2023-B 多波束测线布设官方赛题、附件、结果模板。
- 允许来源：官方竞赛站、主办/承办单位、学校镜像、可信国内镜像。
- 禁止来源：优秀论文、获奖论文、题解、博客、论坛、含最终数值结果说明。
- 保存目录：`official_materials/{年份}_{题号}/problem`、`attachments`、`templates`。
- manifest 输出位置：`official_materials/{年份}_{题号}/manifest.md`。
- 获取失败报告位置：`output/closed_loop/官方材料获取失败_工程优化_2026-07-02.md`。
- 获取失败后动作：选择最高质量 T0/T1/T2 仅做流程测试，或要求人工提供最小材料清单。

## 6. 总控诊断计划

- 输出结构：训练材料理解、材料资格审查、旧题总控诊断、优化专项检查、模型候选与质疑、数据需求、图表计划、人工确认项、评分、标签、闸门。
- 工程优化必查项：评价/目标函数、优化对象、目标、变量、约束、指标、简单方法优先、降维/分区/规则化、对比和敏感性分析。
- 禁止输出内容：不写论文，不写代码，不给最终数值答案。

## 7. 评分与闸门计划

- 评分表：按 15/15/10/20/10/15/10/5 共 100 分。
- P 标签：逐项检查 P1-P10，重点 P3/P4/P5。
- M 标签：逐项检查 M1-M5，M1/M2/M3/M5 阻断 stable。
- 失败复盘触发条件：低于 80、P3/P4/P5、M4 照搬、或错误声明可 stable。
- 修复草案触发条件：失败来自 base/plugin/patch 约束不足。
- 重测任务触发条件：T0/T1/T2、M1/M2/M3/M5、低分或材料补齐后需重跑。
- stable 计数条件：至少 2 道不同题号 T3/T4 通过且无关键 P/M 风险。
