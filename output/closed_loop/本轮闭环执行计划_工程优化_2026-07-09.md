# 本轮闭环执行计划_工程优化_2026-07-09

## 1. 本轮目标

自动完成一轮旧题总控诊断闭环测试，用于检查工程优化提示词体系在独立官方旧题上的稳定性。

本轮只做：

1. 旧题材料扫描。
2. 材料等级与材料风险判断。
3. 总控诊断。
4. 100 分评分。
5. P1-P10 与 M1-M5 检查。
6. 闸门判定。
7. 旧题测试记录与训练日志更新建议。

本轮不做：

1. 不写代码。
2. 不写论文。
3. 不直接给最终答案或最终生产决策方案。
4. 不修改正式 base、plugin、patch。
5. 不正式标记 stable。

## 3. 候选扫描策略

1. 优先选择 T3/T4 官方材料。
2. 若已有两道不同题号 T3 通过，则优先选择机制不同的官方题作为 T4 稳定候选复核。
3. 不读取优秀论文、参考答案、题解或含最终结果的材料。
4. 与当前 patch 来源同题、同附件或同关键结果的题，不计入 stable。
5. 材料缺官方题面、必要附件或必要输出模板时，降级为 T1/T2。

## 4. 本轮候选排序结果

本轮自动扫描后，选择 `2024-B 生产过程中的决策问题`。

选择理由：

1. 来源为官方中文赛题总包。
2. 题面完整，题内表 1、表 2 给出必要参数。
3. 本题无单独附件和官方结果模板要求，不构成附件或模板缺失。
4. 与 A092/A127 定日镜场不同题，与 B311 多波束测线不同机制。
5. 仓库已有 2023-B 与 2024-C 两道 T3 通过记录，本轮适合作为 T4 稳定候选复核测试。
6. 当前只有 Gate 0-2 泛化记录，尚缺按 `02_旧题闭环流` 输出的旧题测试记录。

## 6. 预期判定口径

本轮若满足以下条件，判定总控诊断通过：

1. 总分不低于 80。
2. 不出现 P3、P4、P5。
3. 不出现 M1、M2、M3、M5。
4. 不照搬 2024-C 农业资源配置结构。
5. 不照搬 2023-B 多波束测线空间覆盖结构。
6. 能识别生产过程、质量检测、装配、拆解、退换损失和成本收益权衡。

即使通过，本轮也只建议作为 stable 候选证据，不正式标记 stable。
