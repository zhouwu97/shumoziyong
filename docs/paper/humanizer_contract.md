# 受控语言润色合同

## 目标

受控润色只改善论文表达，不改变数学事实。输入和输出都必须经过保护字段检查；任一保护字段变化，润色结果立即作废并进入人工复核。

## 允许修改

- 调整句式和语序；
- 改善段落衔接；
- 合并重复表达；
- 删除没有信息量的模板化套话；
- 将过度列表化内容改为连贯学术段落。

## 禁止修改

- 数字和百分比；
- 单位、量纲和数量级；
- 公式、变量和上下标；
- 表格内容与图题；
- 文献编号和引用位置；
- 模型、算法和指标名称；
- 结论方向、排序、最优性状态和约束验证结论。

## 执行合同

1. 润色前提取数字、百分比、单位、公式片段、引用编号和锁定术语。
2. 只对普通语言片段改写，保护片段保持逐字一致。
3. 润色后重新提取并逐项比较。
4. 输出机器可读报告，不以“语义基本一致”替代逐项核对。
5. `numbers_changed`、`formulas_changed`、`citations_changed` 或 `units_changed` 任一非空时，`protected_token_check` 必须为 `failed`。

## 人工复核

即使保护字段检查通过，涉及否定词、比较级、因果关系、最优性或敏感性结论的段落仍需人工复核。润色工具不得自动覆盖原论文。
