# 材料风险 M1-M5

M 标签用于判断材料是否会污染旧题闭环。M 标签不一定代表提示词失败，但会影响晋级资格。

| 标签 | 含义 | 处理方式 |
|---|---|---|
| M1 | 答案泄漏：材料来自优秀论文、解析、参考答案或含最终结果复述 | 可跑流程，不计晋级证据 |
| M2 | 附件或结果模板缺失 | 可跑材料受限测试，不计晋级证据 |
| M3 | 同题污染：测试题与 patch 来源论文同题、同附件或同结果格式 | 可跑同题验证，不计晋级证据 |
| M4 | 经验照搬：输出复用了论文特殊公式、布局、数值或结论 | 视为高风险，必要时复盘 |
| M5 | 人工补题意过多：关键约束来自人工总结而非官方题面 | 可跑流程，不计晋级证据 |

## 检查要求

每轮旧题闭环必须输出：

```text
是否存在 M1：
是否存在 M2：
是否存在 M3：
是否存在 M4：
是否存在 M5：
是否允许计入晋级证据：
不计入晋级证据的原因：
```

存在 M1/M2/M3/M5 时，不得计入晋级证据。存在 M4 时，必须说明是否触发 P3、P4 或 P10。
