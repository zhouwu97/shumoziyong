# 材料等级 T0-T4

旧题闭环前必须先判断材料等级。评分通过不等于可计入 stable。

| 等级 | 名称 | 材料来源 | 是否可跑闭环 | 是否可计入 stable |
|---|---|---|---|---|
| T0 | 流程冒烟测试 | 优秀论文、解析、参考答案、从结果反推的题意 | 是 | 否 |
| T1 | 材料受限测试 | 官方题面不完整，缺附件、模板或关键说明 | 是 | 否 |
| T2 | 同题官方题面测试 | 官方题面和附件较完整，但与当前 patch 来源同题或同附件 | 是 | 否 |
| T3 | 独立官方泛化测试 | 独立官方题面、必要附件和输出要求齐全，且不与当前 patch 来源同题 | 是 | 是 |
| T4 | 稳定候选测试 | 已有至少两道不同题号 T3/T4 通过后的复核测试 | 是 | 可建议 |

## 判定原则

- 没有官方题面，不得判为 T3。
- 缺必要附件或输出模板时，不得判为 T3。
- 与训练论文同题、同附件或共享关键结果时，不得计入 stable。
- 来源无法确认官方性时，降级处理。
- T0/T1/T2 可以用于流程验证，但不能证明泛化稳定性。
