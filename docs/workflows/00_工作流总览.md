# 工作流总览

本目录只保留三条主流程入口：

```text
01_论文学习流.md
02_旧题闭环流.md
03_新题执行流.md
```

三条流程不能被含糊地同时启动。唯一允许的组合是 `02_旧题闭环流.md` 的 `production_replay` 显式调用 `03_新题执行流.md` 的生产主链；`diagnostic` 和论文学习流不能冒充生产执行。入口文件只负责调度，不复制 base、plugin、patch 或 rules 的全文。

## 1. 论文学习流

使用场景：日常学习优秀论文。

目标：

```text
优秀论文 PDF
-> 学习卡片
-> 知识卡片 JSON
-> patch 草案
-> patch_index 实际登记
-> 优秀论文评审标准注册表登记
-> training_log 实际更新
```

边界：

- 不修改正式 `prompt_base/`。
- 不修改正式 `prompt_plugins/`。
- 不标记 `competition_evidenced`。
- 不把单篇论文经验当通用规则。
- 单篇或未核验经验只能进入候选层；全局规则升级必须有核验和跨论文或跨题证据。

## 2. 旧题闭环流

使用场景：赛前诊断，或用官方旧题执行无泄漏的真实生产回归。

目标：

```text
diagnostic：材料资格 → 总控诊断 → Prompt/失败标签检查
production_replay：官方材料 → 03 新题生产主链 → Paper Admission → 独立评审
```

边界：

- `diagnostic` 不写代码、论文和最终答案，也不构成建模能力证据。
- `production_replay` 必须完成模型、代码、结果、图表和完整论文，且不得读取同题答案或旧结果。
- 不未经人工确认修改正式 base/plugin/patch。
- `validated_candidate` 只能由无泄漏 production replay 或陌生题完整生产支持。

## 3. 新题执行流

使用场景：比赛当天或模拟赛当天。

目标：

```text
新题
-> 题型识别与 LEARNING_CONTEXT
-> 题面拆解与建模草案
-> R1 建模审核（编码前）
-> 可复现代码、结果与图表
-> R2 实验复现审核
-> 单一 Typst/LaTeX 源稿论文初稿
-> R3 论文逻辑审核
-> ENGINEERING_VERIFICATION / Ledger / PDF / package-draft
-> R4 格式与提交审核
-> 每问 PAPER_ADMISSION
-> 作者侧创新性/可读性/AI 痕迹自检
-> 编排器生成 R5 review_request.json
-> 配置 Reviewer adapter 后创建独立 R5 全面盲审对话
-> 修补 MUST 后生成新复审请求并由 adapter 创建新对话
-> 完成附件核包后派生 SUBMISSION_STATUS
```

边界：

- 默认连续执行，只在会改变数学问题的关键歧义上暂停。
- 验证强度随题型和风险调整，不固定额外 Schema、Reviewer 哈希或 Evidence 数量。
- 论文必须由一个 `main.typ` 或 `main.tex` 源稿编译，禁止另写短正文生成 PDF。
- Formal Result、Seal、Collector 和多席隔离评阅只用于显式 `formal_audit`。
- Final Reviewer 采用优秀论文学习库抽象出的国赛标准，不读取作者对话或同题答案。
- “AI率”不作为事实；只报告可定位证据支持的低/中/高 AI 痕迹风险。
- 工程 PASS、论文准入、Reviewer 推荐和提交就绪是四个不同结论，任何前一层都不能替代后一层。
- 页数不设硬门槛；每问模型、求解、结果、验证、解释和边界必须有证据定位。
- 历史论文经验必须先经过注册表权限过滤，再在作者侧记录计划用途和实际论文落点；不得等到 Reviewer 才首次使用。

## 4. 文件职责

| 位置 | 职责 |
|---|---|
| `README.md` | 项目定位、目录说明、三个入口 |
| `docs/workflows/` | 三条流程的调度入口 |
| `docs/workflows/rules/` | 旧题闭环的判定规则 |
| `prompt_base/` | 通用总控诊断规则 |
| `prompt_plugins/` | 题型专项规则 |
| `prompt_patches/` | 单篇论文经验补丁 |
| `prompt_patches/patch_index.json` | patch 检索、筛选和 runtime 导出索引 |
| `papers/templates/` | 论文学习卡片和知识卡片模板 |
| `papers/EXCELLENT_PAPER_REVIEW_STANDARD_REGISTRY.json` | 优秀论文来源、核验状态与评审规则晋级事实源 |
| `training_log.md` | 训练和测试事实记录 |
